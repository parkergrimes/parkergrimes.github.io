### GitHub Shortcuts

### Site wide shortcuts

Command | Description
-- | --
S|Focus command bar, search all repositories
/|Focus command bar, sharch the current repository
G N|Go to Notifications
?|Bring up help dialog
J|Move selection down
K|Move selection up
X|Toggle selection
O / Enter|Open selection

### Repositories

Command | Description
--|--
G C|Go to Code
G I|Go to Issues
G P|Go to Pull Requests
